# leonardomaier/kaeru

The Kaeru Command Line Interface (CLI)

### Usage

```bash
Kaeru CLI [Node: > 12.22.1]

Commands:
  kaeru new newconfig.yaml Creates configuration file
  kaeru run-command <command> --config <config>
```
